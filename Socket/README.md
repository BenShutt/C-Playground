# Socket

A simple socket TCP server written in C.

## Usage

Shorthand to clean, make, and run the program:

```bash
./scripts/run.sh
```

## IP Address

To get the wireless IP:

```bash
ipconfig getifaddr en0
```

The server will connect to port `666`.
